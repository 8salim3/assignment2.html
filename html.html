TERM 12

Hypertext Markup Language

The standard language used to create and structure content on the World Wide Web.

What is HTML?

HTML (Hypertext Markup Language) is the foundational language of the Web. It uses a system of elements represented by tags to structure content such as headings, paragraphs, links, images, tables, and forms on a web page.

HTML was created by Tim Berners-Lee in 1991. The current standard is HTML5, which introduced powerful features like video, audio, canvas, and semantic elements.

Basic Structure of an HTML Page

  • <!DOCTYPE html> — declares the document type
  • <html> — the root element of the page
  • <head> — contains metadata, title, and links to CSS
  • <body> — contains all visible page content

Common HTML Elements

  • <h1> to <h6> — headings from largest to smallest
  • <p> — paragraph of text
  • <a href=""> — hyperlink to another page or resource
  • <img src=""> — embeds an image
  • <ul> and <ol> — unordered and ordered lists
  • <div> — a generic container for grouping elements
  • <form> — creates an interactive form for user input

HTML, CSS and JavaScript

💡 HTML defines the structure of a web page. CSS controls the appearance (colors, fonts, layout). JavaScript adds interactivity (animations, dynamic content). Together, these three form the core of every modern website.

Why HTML Matters

Every website you have ever visited is built on HTML. It is the skeleton of the Web — without it, there would be no way to structure or display content in a browser. Learning HTML is the essential first step in web development.
