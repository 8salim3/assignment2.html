# Internet Terminology Website

A multi-page website covering 16 essential Internet and Web terminology terms, built as part of a university assignment.

## Topics Covered

1. World Wide Web (WWW)
2. TCP/IP
3. Web Browsers
4. Web Servers
5. Uniform Resource Locator (URL)
6. Domain Name Server (DNS)
7. Hypertext Transfer Protocol (HTTP)
8. Intranet
9. Extranet
10. Multitier Architecture
11. File Transfer Protocol (FTP)
12. Hypertext Markup Language (HTML)
13. Web 1.0
14. Web 2.0
15. Web 3.0
16. Web 4.0

## Project Structure

- `index.html` — Home page with links to all 16 terms
- `style.css` — Shared stylesheet for all pages
- `images/` — Folder containing all images used across the site
- Individual HTML files for each terminology term

## How to View

Open `index.html` in any web browser to start exploring the site.
