TERM 04

Web Servers

Hardware or software systems that store, process, and deliver web pages to users upon request over the Internet.

What is a Web Server?

A web server is a system — either hardware, software, or both — that responds to requests from web browsers and delivers web content such as HTML pages, images, videos, and files.

Every website you visit is hosted on a web server somewhere in the world. When you type a URL, your browser communicates with that server to retrieve and display the page.

How a Web Server Works

The process follows a simple request-response model using HTTP or HTTPS. The browser sends a request, the server processes it, locates the requested resource, and sends it back to the browser for display.

💡 A single physical server machine can host thousands of websites simultaneously using a technique called virtual hosting.

Popular Web Server Software

  • Apache HTTP Server — the most widely used web server software globally
  • Nginx — known for high performance and handling large traffic
  • Microsoft IIS — used mainly in Windows server environments
  • LiteSpeed — popular for WordPress hosting due to its speed

Types of Web Servers

  • Static web server — serves fixed content that does not change
  • Dynamic web server — generates content on the fly using databases and scripts
  • Proxy server — acts as an intermediary between the client and another server

Why Web Servers Matter

Web servers are the foundation of the Internet. Without them, there would be no websites, no online shopping, no streaming, and no cloud services. They handle billions of requests every single day across the globe.
