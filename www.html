TERM 01

World Wide Web

A global system of interconnected hypertext documents and resources, navigated via the Internet using a web browser.

What is the World Wide Web?

The World Wide Web (WWW), commonly known as "the Web," is an information system that enables the sharing and access of documents and resources through the Internet. It was invented by British scientist Tim Berners-Lee in 1989 while working at CERN.

The Web operates through a system of hyperlinks and URLs that connect web pages together, allowing users to navigate from one document to another seamlessly across the globe.

How It Works

When you type a web address into your browser, the browser sends a request to a web server. The server responds by sending back the requested web page, which your browser then displays. This entire process happens in milliseconds.

💡 The Web is NOT the same as the Internet. The Internet is the global network infrastructure, while the Web is one of many services that runs on top of it.

Key Components

  • HTML — the language used to create web pages
  • HTTP/HTTPS — the protocol used to transfer web data
  • URLs — the addresses used to locate web resources
  • Web Browsers — software used to access and display web pages
  • Web Servers — computers that host and serve web content

Importance of the WWW

The Web has revolutionised how humanity communicates, conducts business, accesses education, and shares information. From e-commerce to social media, nearly every modern digital service is built on top of the World Wide Web.
