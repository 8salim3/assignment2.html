Topic 1 — Internet Terminology

Understanding the
Building Blocks
of the Internet

A comprehensive reference guide covering 16 essential Internet and Web terminology terms. Click any term below to explore its full explanation.

TERM 01

World Wide Web

The system of interlinked hypertext documents accessed via the Internet.
TERM 02

TCP/IP

The foundational communication protocols that power the Internet.
TERM 03

Web Browsers

Software applications used to access and navigate the World Wide Web.
TERM 04

Web Servers

Systems that store and serve web content to users on request.
TERM 05

URL

Uniform Resource Locator — the address of a resource on the Internet.
TERM 06

Domain Name Server

The Internet's phonebook — translates domain names to IP addresses.
TERM 07

HTTP

Hypertext Transfer Protocol — the rules for transferring web data.
TERM 08

Intranet

A private internal network used within an organisation.
TERM 09

Extranet

A controlled private network shared with external authorised users.
TERM 10

Multitier Architecture

A design pattern that separates applications into distinct layers.
TERM 11

File Transfer Protocol

A standard protocol for transferring files between computers.
TERM 12

HTML

Hypertext Markup Language — the standard language for web pages.
TERM 13

Web 1.0

The first generation of the Web — static, read-only pages.
TERM 14

Web 2.0

The social, interactive and dynamic era of the Web.
TERM 15

Web 3.0

The semantic and decentralised next generation of the Web.
TERM 16

Web 4.0

The intelligent, symbiotic web powered by AI and IoT.
